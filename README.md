# DISO
DISO (Detect &amp; Inspect Synthetic Output)
DISO es una herramienta creada para reconocer si una imagen o un video fue hecho por inteligencia artificial o si es real.
Analiza cada detalle del contenido, detecta patrones que dejan las IA y compara la información para decirte si el archivo es auténtico o generado por una máquina.
